Heroku buildpacks

When you git push heroku, Heroku’s slug compiler prepares your code for execution by the Heroku dyno manager. At the heart of the slug compiler is a collection of scripts called a buildpack.

Meteorite buildpack

https://github.com/oortcloud/heroku-buildpack-meteorite

Putting Telescope on Heroku

						
git clone https://github.com/TelescopeJS/Telescope.git

						
heroku apps:create  --stack cedar --buildpack https://githubcom/oortcloud/heroku-buildpack-meteorite.git

						
heroku config:add ROOT_URL=http://.herokuapp.com

						
git push heroku master
